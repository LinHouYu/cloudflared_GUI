# Cloudflare Tunnel GUI


一个基于 **Python + Tkinter** 的图形化界面工具，用于简化 **Cloudflare Tunnel (cloudflared)** 的使用。  
支持服务端、客户端配置，主题切换，日志查看，以及一键更新/登录等功能。

## ✨ 功能特性
- **多标签页界面**：服务端、客户端、杂项功能分区清晰  
- **主题切换**：支持浅色/深色主题，自动保存上次选择  
- **Cloudflared 管理**：
  - 一键登录授权（自动复制认证链接到剪贴板）
  - 检查版本 / 更新 / 跳转下载页面
- **日志输出**：实时显示命令执行结果  
- **打赏支持**：内置微信、USDT 二维码展示  
- **个性化设置**：自动记忆上次打开的标签页
